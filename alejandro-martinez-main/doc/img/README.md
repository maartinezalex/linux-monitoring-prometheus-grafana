# Sobre esta carpeta:

### `doc/img/`

En esta carpeta pon todas las imágenes a las que hagas referencia en
tu proyecto: figuras, diagramas, gráficos etc.

Lo puedes generar con cualquier software, y dejas aquí la
imagen. Luego la puedes enlazar a tu documento, pero tendrías en un
lugar todos los diagramas, figuras e imágenes.

**IMPORTANTE**: Se recomienda utilizar el formato `PNG` y que no  ocupen más de  `1 MB`
